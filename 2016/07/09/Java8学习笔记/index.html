#Java8学习笔记

1. lambda表达式

  • lambdas可以引用类的成员变量和局部变量(如果不是final类型，会隐士的转换成final)

eg

和

Arrays.asList( “a”, “b”, “d” ).sort( ( e1, e2 ) -> e1.compareTo( e2 ) );

Arrays.asList( “a”, “b”, “d” ).sort( ( e1, e2 ) -> {
int result = e1.compareTo( e2 );
return result;
} );
```

2. 函数式接口

接口只有一个方法，这个方法会自动的隐士转换成lambda表达式

  • 使用@FunctionInterface可以将一个接口定义成函数式接口
  • 在函数式接口中可以存在使用default修饰的方法
