Java8学习笔记

Posted on 2016-07-07   |   In Java 8   |  

This is a continuous updating learning notes for Java8

1. 函数接口

接口只有一个方法，这个方法会自动的隐式转换成lambda表达式

  • 函数接口是只有一个抽象方法的接口， 用作lambda表达式的类型
  • 如果要将lambda表达式定义成一个接口类型，则要将lambda表达式的类型和该接口中的唯一一个抽象方法的签名保持一致
  • 使用@FunctionInterface可以将一个接口定义成函数式接口
  • 在函数式接口中可以存在使用default修饰的方法
1.1 jdk提供的常用的函数接口
接口 参数 返回类型
Predicate<T> T boolean
Consumer<T> T void
Function<T> T R
Supplier<T> None T
UnaryOperator<T> T T
BinaryOperator<T> (T,T) T

2. 惰性求值方法/及早求值方法

  • 只描述Stream最终不产生新的集合的方法叫做惰性求值方法, 例如filter方法
  • 最终会从Stream产生值的方法叫做及早求值方法, 例如count方法

注意: 判断一个操作是惰性求值还是及早求值只需要看它的返回值, Stream->惰性求值, void/另一个具体的值->及早求值

  • Stream中的max和min方法用于获取指定比较方式下的数据中的最大值和最小值

注意: min和max的参数是Comparator,jdk1.8中, Comparator提供了comparing方法用来指定比较大小的数据
