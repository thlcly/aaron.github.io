Jackson学习笔记

Posted on 2016-07-07   |   In Java Web   |  

This is a continuous updating learning notes for Jackson

1. Jackson处理json的方式

  • Stream API: JsonParser读取json数据, JsonGenerator生成json数据, 在三种方法中最高效
  • Tree Model: 在内存中生成一个json文档, 使用ObjectMapper操作json, 在三种方法中最灵活
  • Data Binding: 使用java对象生成Json, 使用ObjectMapper操作json的读写

2. JSON type 和 Java type 映射关系

json type java type
object LinkedHashMap
array ArrayList
string String
number (整数) Integer, Long or BigInteger (smallest applicable)
number(浮点数) Double (configurable to use BigDecimal)
true \ false Boolean
