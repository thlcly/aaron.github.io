This is a continuous updating learning notes for Git

1. git config配置

  • 开启git的大小写敏感

    git config --global core.ignorecase false

  • 查看git全局配置

    git config --global --list

  • git配置log信息样式

    git log --graph --pretty=format:'\''%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset'\'' --abbrev-commit --date=relative

2. git init命令

git init命令主要是创建了4个文件夹

  • .git
  • .git/refs/heads, .git/refs/tags: 存储了本地分支, tag, 如果关联的远程, 则还会有一个remotes文件夹, 用于存放远程分支
  • .git/objects: 存储了所有的数据内容
  • .git/HEAD(文件): 指向当前分支
